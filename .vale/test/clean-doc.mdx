## How to set up analytics

PostHog is an analytics platform built for developers. It helps you understand how people interact with your product.

You can use Feature Flags, Session Replay, and Product Analytics to make data-driven decisions. Check out Error Tracking and Data Warehouse too.

When developers log in, they can see their dashboard. They can customize their profile settings and organize their projects.

For more information, check out the [PostHog documentation](https://posthog.com/docs) or the [getting started guide](https://posthog.com/tutorials).

PostHog – the best analytics platform – helps you ship better products.

The platform supports events, actions, and cohorts for segmenting your data.

Add your project token to the initialization code. You'll also need a personal API key for server-side access.
