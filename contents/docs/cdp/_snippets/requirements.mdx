## Requirements

Using this conector requires either PostHog Cloud with the [data pipeline add-on](https://us.posthog.com/organization/billing), or a self-hosted PostHog instance running [version 1.30.0](/blog/the-posthog-array-1-30-0) or later.

Self-hosting and not running 1.30.0? Find out [how to update your self-hosted PostHog deployment](/docs/runbook/upgrading-posthog).
