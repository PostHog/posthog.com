The Paddle connector can link customers, discounts, prices, products, subscriptions, transactions, and adjustments to PostHog.

To link Paddle:

1. Go to the [sources tab](https://app.posthog.com/data-management/sources) of the data pipeline section in PostHog.

2. Click **+ New source** and then click **Link** next to Paddle.

3. Next, you need an API key from Paddle. In your Paddle dashboard, go to **Developer tools** > **Authentication** and create a new API key. Copy the API key value (it starts with `pdl_live_` for production or `pdl_sbox_` for sandbox).

4. Back in PostHog, paste the API key in the `API key` field and click **Next**. Optionally, add a prefix to your table names.

5. Select the schemas you want to sync and configure the sync frequency. Once done, click **Import**.

Once the syncs are complete, you can start using Paddle data in PostHog.

> Incremental syncs are supported for the transactions table using the `billed_at` field. Other tables perform full syncs.
