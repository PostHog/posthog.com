The Linear connector syncs issues, projects, teams, users, comments, labels, cycles, and issue relations to PostHog.

To link Linear:

1. Go to the [sources tab](https://app.posthog.com/data-management/sources) of the data pipeline section in PostHog.

2. Click **+ New source** and select Linear.

3. Select the Linear workspace you want to connect and authorize the connection.

4. *Optional:* Add a prefix to your table names.

5. Select the tables you want to sync and configure the sync frequency.

6. Click **Import**.

Once the sync is complete, you can query your Linear data in PostHog.
