## Open source

PostHog Code is open source under the MIT license. The full source code lives in the [PostHog/code GitHub repo](https://github.com/PostHog/code). More about the license and contribution guidelines in the [PostHog Code open source](/docs/posthog-code/open-source) page.

## Platform availability

The macOS app is maintained by the PostHog Code team. The Windows app is community maintained and reviewed by the PostHog Code team.

For bugs, search [open GitHub issues](https://github.com/PostHog/code/issues) first. If nothing matches, [open a new issue](https://github.com/PostHog/code/issues/new) with your OS, app version, and steps to reproduce.
