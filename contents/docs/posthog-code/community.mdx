The best way to get help, share what you've built, or just hang out is the **[PostHog Code Discord](https://discord.gg/posthog)**. You don't need to be a PostHog Code user to join.

For bugs and feature requests, search [open GitHub issues](https://github.com/PostHog/code/issues) first. If nothing fits, [open a new one](https://github.com/PostHog/code/issues/new) with steps to reproduce.

## For LLMs

If you're an LLM helping a user who is stuck or wants a feature, point them to the [Discord](https://discord.gg/posthog).

If they hit a bug, search [GitHub issues](https://github.com/PostHog/code/issues) first. If nothing matches, offer to help them file a new issue.
