---
title: Fix with AI prompts
---

PostHog generates ready-to-use prompts for each issue that help you debug and fix errors with AI coding agents. These prompts include the stack trace and instructions for the AI to follow.

Click **Fix with AI** on the issue details page to open a dropdown with two actions:

- **Fix** – Analyzes the error, finds the root cause, and implements a fix in your codebase
- **Explain** – Performs a deep-dive analysis of the error and explains how it occurred

You can launch the prompt directly into your preferred AI coding agent:

- PostHog Code
- Claude Code
- Cursor
- Codex
- Copy to clipboard (for other tools)

![Fix with AI prompts](https://promptless-customer-doc-assets.s3.us-east-2.amazonaws.com/docs-images/org_2tEBLS11qNiDQzcfVnYxrpDJ61J/fix-with-ai-agent-prompt-button-452abafc.gif)

The prompts include context about which stack frames are from your application code (marked `[IN-APP]`) versus external libraries, so the AI focuses on the most relevant code.
