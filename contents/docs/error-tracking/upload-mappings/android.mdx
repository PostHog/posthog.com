<CalloutBox icon="IconInfo" title="[AGP 8](https://developer.android.com/build/releases/gradle-plugin) or later" type="fyi">

Automatic mappings uploading is handled through the Gradle build process on Android.

Install the [PostHog Android Gradle Plugin](https://github.com/PostHog/posthog-android/blob/main/posthog-android-gradle-plugin/CHANGELOG.md).

```kotlin
// Available through mavenCentral()
plugins {
    id("com.posthog.android") version "$version"
    ...
}
```
