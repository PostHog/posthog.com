To install the chart using [Helm](https://helm.sh/) with the release name `posthog` in the `posthog` namespace, run the following:

```shell
helm repo add posthog https://posthog.github.io/charts-clickhouse/
helm repo update
helm upgrade --install -f values.yaml --timeout 30m --create-namespace --namespace posthog posthog posthog/posthog --wait --wait-for-jobs --debug
```

Note: if you decide to use a different Helm release name or namespace, please keep in mind you might have to change several values in
your `values.yaml` in order to make the installation successful. This is because we build several Kubernetes resources
(like service names) using those.
