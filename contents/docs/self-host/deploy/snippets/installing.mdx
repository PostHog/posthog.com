To install the chart using [Helm](https://helm.sh/) with the release name `posthog` in the `posthog` namespace, run the following:

```shell
helm repo add posthog https://posthog.github.io/charts-clickhouse/
helm repo update
helm upgrade --install -f values.yaml --timeout 20m --create-namespace --namespace posthog posthog posthog/posthog --wait --wait-for-jobs --debug
```
