To install the chart using [Helm >= v3](https://helm.sh/) with the release name `posthog` in `posthog` namespace, run the following:

```shell
helm repo add posthog https://posthog.github.io/charts-clickhouse/
helm repo update
helm install -f values.yaml --timeout 20m --create-namespace --namespace posthog posthog posthog/posthog
```

<details>
<summary>Troubleshooting</summary>

If you don't see some pods come up (e.g. `chi-posthog-posthog-0-0-0` pod is missing after deleting a previous instance and trying to install PostHog again), try running `helm upgrade -f values.yaml -n posthog posthog posthog/posthog`.
</details>
