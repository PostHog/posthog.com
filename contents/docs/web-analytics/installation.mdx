Web analytics enables you to easily track and monitor many of the most important metrics for your website. 

Unlike [product analytics](/docs/product-analytics), web analytics offers a more streamlined and focused experience. This is especially useful for marketers, content creators, or anyone used to tools like [Google Analytics](/blog/google-analytics-to-posthog). It includes a [dashboard](/docs/web-analytics/dashboard) that offers essential metrics, such as visitors, views, sessions, session duration, bounce rate, [conversions](/docs/web-analytics/conversion-goals), paths, referrers, and more.
