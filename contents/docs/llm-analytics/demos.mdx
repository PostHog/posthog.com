Explore real-world examples of LLM analytics in action. These projects demonstrate different ways to integrate PostHog's LLM analytics features.

**[Browse all LLM analytics demos &rarr;](/side-projects?tag=llm-analytics)**

## Example projects

### Multi-provider integrations

- **[LLM Analytics Apps](/side-projects/llm-analytics-apps)** - Interactive CLI showcasing PostHog LLM analytics across 13+ AI providers (Anthropic, OpenAI, Google, LangChain, and more). Great for understanding how different SDKs integrate with PostHog.

### Production applications

- **[Andy's Daily Factoids](/side-projects/andys-daily-factoids)** - Full-stack AI factoid generator demonstrating LangChain callbacks, multi-platform observability, cost tracking, and real-time streaming. [Live demo](https://andys-daily-factoids.com)

- **[Anomstack](/side-projects/anomstack)** - Anomaly detection system with LLM-powered analysis. Shows how to combine PostHog metrics with AI-based anomaly explanation. [Live demo](https://anomstack-demo.fly.dev)

### AI assistants & agents

- **[GitHub Standup Agent](/side-projects/github-standup-agent)** - Multi-agent standup summarizer using OpenAI Agents SDK. Demonstrates automatic agent tracing with `posthog.ai.openai_agents.instrument()`.

- **[Repo Stalker](/side-projects/repo-stalker)** - GitHub PR/Issue AI assistant with custom span tracking for different interaction types (chat, summaries, analysis).

- **[Plot Agent](/side-projects/plot-agent)** - AI data visualization assistant using LangGraph. Shows PostHog callback integration with conversation session tracking.

- **[Anomaly Agent](/side-projects/anomaly-agent)** - LLM-powered anomaly detection for time series data using LangGraph. Demonstrates multi-step AI workflows with verification.

### PostHog demos

- **[HogFlix (PostHog Demo 3000)](/side-projects/posthog-demo-3000)** - Full PostHog feature showcase including LLM chat with both LangChain and direct OpenAI integration patterns.

## Integration patterns demonstrated

| Pattern | Example Projects |
|---------|-----------------|
| LangChain Callbacks | Andy's Daily Factoids, HogFlix, Plot Agent, Anomaly Agent |
| OpenAI Agents SDK | GitHub Standup Agent |
| Direct PostHog SDK | LLM Analytics Apps |
| Custom HTTP Capture | Repo Stalker |
| Multi-platform Observability | Andy's Daily Factoids (PostHog + Braintrust + LangSmith) |
