---
title: Prompt Management
---

Prompt Management lets you author, version, and ship LLM prompts and reusable agent skills without code deploys. Iterate on prompts in the PostHog UI and fetch them at runtime through the SDK.

## In this section

- [Prompts](/docs/prompt-management/prompts) – Create, version, and serve LLM prompts to your application at runtime.
- [Skills store](/docs/prompt-management/skills-store) – Centralized, versioned storage for reusable agent skills that work across any MCP-connected agent.

## Related products

- [AI Observability](/docs/ai-observability) – Capture traces and generations from your LLM application.
- [Evaluations](/docs/ai-evals) – Score the quality of generations produced by your prompts.

## Installation

Prompts are fetched through the PostHog SDK. If you haven't installed the SDK yet, follow the [AI Observability installation guide](/docs/ai-observability/installation) — the same setup powers prompt fetching.
