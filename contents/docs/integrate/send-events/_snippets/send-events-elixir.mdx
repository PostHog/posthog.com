```elixir
Posthog.capture("user_signed_up", %{
  distinct_id: distinct_id_of_the_user,
  properties: %{
    login_type: "email",
    is_free_trial: true
  }
})
```

### Batching events

To capture multiple events at once, use `batch()`:

```elixir
Posthog.batch([{"user_signed_up", [distinct_id: distinct_id_of_the_user], nil}])
```
