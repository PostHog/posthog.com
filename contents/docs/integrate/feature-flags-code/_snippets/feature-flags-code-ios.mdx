### Boolean feature flags

```swift
if (PostHogSDK.shared.isFeatureEnabled("flag-key")) {
    // Do something differently for this user
}
```

### Multivariate feature flags

```swift
if (PostHogSDK.shared.getFeatureFlag("my-flag") as? String == "variant-key") { // replace "variant-key" with the key of your variant
    // do something
}
```

### Reloading feature flags

Feature flag values are cached. If something has change with your user and you'd like to refetch their flag values, call `PostHogSDK.shared.reloadFeatureFlags()`.
