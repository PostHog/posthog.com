1. Open [v0](https://v0.app/chat/settings/mcp-connections) settings and navigate to **Connections**.
2. Click **Add MCP** and select **Custom MCP**.
3. In the form, name the MCP `PostHog` and use the URL `https://mcp.posthog.com/mcp`.
4. Under **Authentication**, select **OAuth** and click **Authorize** to kick off the OAuth flow.
5. When prompted, log in to PostHog to authorize access. Then, you can access the PostHog MCP server in your v0 chat.
