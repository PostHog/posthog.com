[PostHog AI](/docs/posthog-ai/overview) can help you understand feature flag performance, answer questions about flag configurations, and create new flags — all from the in-app chat.

## What PostHog AI can do with feature flags

When working with feature flags, PostHog AI can:

- **Create feature flags** directly from a natural language description using the `create_feature_flag` tool.
- **Analyze flag performance** by querying evaluation data, rollout percentages, and user breakdowns.
- **Answer configuration questions** about release conditions, targeting rules, and flag variants.
- **Write HogQL queries** to dig into flag evaluation data, such as which users are seeing which variants.
- **Explain PostHog features** based on our documentation, like how to set up flags, bootstrapping, or dependencies.

## Example prompts

Try these prompts in the [PostHog AI chat](/docs/posthog-ai/platform-and-chat-ui):

```
- How many users are seeing the new-dashboard feature flag?
- Show me the rollout percentage for my feature flag.
- Which user segments are enabled for my flag?
- Write a query to analyze feature flag performance over the last 7 days.
- How do I set up a feature flag?
```

> **Note:** PostHog AI can also answer general questions about PostHog features based on our documentation. If you're unsure how something works, just ask.

## Getting started

To start using PostHog AI with feature flags, follow the [getting started guide](/docs/posthog-ai/start-here). Once set up, open the PostHog AI chat from anywhere in the app and start prompting.

For a full list of PostHog AI tools and capabilities, see the [tools reference](/docs/posthog-ai/tools).
