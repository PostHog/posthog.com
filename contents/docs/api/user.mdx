In PostHog, there is an API endpoint available to get all information on your own user and team.
This includes all possible event names, event properties, API key, app URLs, current PostHog version and more.


## List current user

```shell
GET /users/@me/
```

Example request:

```bash
curl https://posthog.example.com/api/users/@me/
```

## `user` - deprecated

This endpoint has been deprecated since April 5th, 2021 and support for it will be removed in the future.
