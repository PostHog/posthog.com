Our annotations endpoints allows you to create, update, and fetch annotations from PostHog.

## Endpoints

<Endpoint endpoint='/api/annotation' allowedMethods={['get', 'post']} />

<Endpoint endpoint='/api/annotation/<annotation_id:int>' allowedMethods={['get', 'put', 'patch', 'delete']} />

## Examples

### Get all annotations

#### Request

```
GET /api/annotation
```

##### Optional Query Parameters

| Parameter | Valid Values | 
| --- | --- | 
| `scope` | `organization` or `dashboard_item` | 
| `deleted` | `true` or `false` | 
| `order` | `updated_at`, `created_at`, or `date_marker`.   &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; You can also add a `-` in front to reverse the order e.g. `-updated_at`. |

### Create an annotation

#### Request

```
POST /api/annotation
```

##### Request Body

| Field | Type / Valid Values | Required |
| --- | --- | --- | 
| `scope` | `organization` or `dashboard_item` | ✅ |
| `content` | `string` | ✅ |
| `date_marker` | ISO timestamp string | ✅ | 
| `deleted` | `boolean` | ❌ | 
| `dashboard_item` | `integer` | ❌ |
