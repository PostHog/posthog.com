This page will cover:

- Overview of supported query types
- **HogQL queries**:
  - Direct SQL-like queries
  - Support `variables` parameter for runtime substitution
  - Full flexibility for custom queries
- **Insight queries**:
  - TrendsQuery
  - FunnelsQuery
  - LifecycleQuery
  - RetentionQuery
  - PathsQuery
  - StickinessQuery
  - Support `query_override` parameter for customization
- Key difference: HogQL uses `variables`, Insight queries use `query_override`
- When to use each query type
- Converting insights to endpoints
