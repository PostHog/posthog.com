Kea is a state management library for React that provides a batteries-included approach with Redux, Redux-Saga, and Redux-Thunk integration. Named after the highly intelligent New Zealand parrot, Kea offers a production-ready state management solution used by PostHog and other companies. Now in v3 with improved TypeScript support and developer experience.
