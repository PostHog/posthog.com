A terminal UI for managing AWS resources, inspired by lazygit, k9s, and lazydocker. Features EC2 instance management with SSM session support, S3 bucket browsing with in-editor file editing, and EKS cluster management with k9s integration. VIM-style navigation and command interface. Built with Bubble Tea and AWS SDK for Go.
