Apple's SF Mono font enhanced with Powerline glyph support through Nerd Fonts patching. Enables styled terminal statuslines and shell prompts for tools like Vim, Shellder, and Starship across macOS, Linux, and Windows. Includes 14 different font variants from Light to Heavy, with italic combinations. Install via Homebrew or manual download.
