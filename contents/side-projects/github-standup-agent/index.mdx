An AI-powered CLI tool that generates daily standup summaries from GitHub activity using the OpenAI Agents SDK. Automatically collects PRs, issues, commits, and code reviews, then uses multi-agent AI to create concise, well-formatted standup summaries.

## What it does

- **Multi-Agent Workflow**: Coordinator Agent orchestrates Data Gatherer and Summarizer agents
- **GitHub Integration**: Collects PRs, issues, commits, and code reviews via GitHub CLI
- **Interactive Chat**: Refine summaries through conversation
- **Session Persistence**: Resume previous sessions with SQLite storage
- **Style Customization**: Configure output format via style.md and few-shot examples
- **Copy to Clipboard**: Quick export of generated summaries

## Architecture

- **Coordinator Agent** (gpt-4o): Orchestrates the workflow
- **Data Gatherer Agent** (gpt-4o-mini): Collects GitHub data via `@function_tool` decorated functions
- **Summarizer Agent** (gpt-4o): Creates the final standup summary
- **Built with**: OpenAI Agents SDK, Typer CLI framework, SQLite for persistence

## How it uses PostHog LLM Analytics

The agent demonstrates automatic OpenAI Agents SDK instrumentation:

```python
from posthog.ai.openai_agents import instrument

# Automatic agent tracing when POSTHOG_API_KEY is set
instrument()
```

**Tracked events**:
- Full agent trace tracking with privacy mode control
- Custom events: `standup_generated`, `standup_saved`
- Summary metadata, user info, execution metrics
- Debug logging via `POSTHOG_DEBUG` environment variable

**Configuration**:
```bash
POSTHOG_API_KEY=phc_...
POSTHOG_HOST=https://us.posthog.com
POSTHOG_DISTINCT_ID=your-username
```

## Try it yourself

Install from PyPI:

```bash
pip install github-standup-agent
github-standup-agent
```

Or run from source:

```bash
git clone https://github.com/andrewm4894/github-standup-agent
cd github-standup-agent
pip install -e .
github-standup-agent
```

All data stays local - only API calls are made externally.
