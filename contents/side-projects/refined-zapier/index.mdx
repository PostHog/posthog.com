An open-source browser extension inspired by Refined GitHub that enhances the Zapier automation platform's user experience. Features include folder searching with filtering, commit messages for tracking zap changes, prevention of unnamed zaps being published, detailed commit history viewing, and enhanced styling with Poppins fonts and TailwindCSS. Available for Chrome, Edge, and Opera.
