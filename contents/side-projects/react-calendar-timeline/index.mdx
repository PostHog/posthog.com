A contemporary React component for displaying interactive timeline visualizations. Features a full TypeScript rewrite with React 18+ support, drag-and-drop item manipulation with configurable snapping, customizable headers and markers, and left/right sidebar support. Perfect for project management, scheduling, and resource planning applications.
