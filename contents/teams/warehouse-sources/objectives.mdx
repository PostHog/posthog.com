
Objective: New Sources + source management

* **Key Results:**
    * New sources for a bunch of popular saas products and databases
    * Source self-healing system - use AI to open PRs with fixes to sources based on error tracking
    * A new interface for debugging source/pipeline issues from support
    * Improved UX for listing new sources in-app


Objective: Re-architecture + run improvements

* **Key Results:**
    * Re-architecture of the pipeline to be more event-driven
    * Better alerting, monitoring, and documentation of the pipeline/sources/runs, both internally and for users, to allow for self-debugging of issues without support
    * Writing out to ducklake or an equivalent system for the new query engine


Objective: Improved source config

* **Key Results:**
    * Visibility of primary keys and incremental field indexes
    * Configure columns to import from the source
    * Multi-database/schema support for database sources
    * Postgres logical replication
