## The brand at a glance

| Element      | PostHog                                | Not PostHog                                  |
| ------------ | -------------------------------------- | -------------------------------------------- |
| Voice        | Direct, specific, conversational       | Corporate, vague, buzzword-heavy             |
| Visual style | Handcrafted, slightly weird, nostalgic | Gradient-heavy, generic SaaS, abstract blobs |
| Humor        | Genuine, specific, can be dark         | Forced, try-hard, "ooh a hedgehog"           |
| Design goal  | Look intentional                       | Look expensive                               |
| Personality  | Opinionated, slightly weird, honest    | Polished but hollow                          |

## Design checklist

Before you ship anything:

- [ ] Remove the logo – does it still feel like PostHog?
- [ ] Is the message clear without explanation?
- [ ] Is every design element there for a reason?
- [ ] Does it feel human?
- [ ] Would someone enjoy looking at this?
- [ ] Would a developer trust this?
- [ ] Does the copy avoid hedge words and buzzwords?
- [ ] If it has humor, is it actually funny?
- [ ] Would it survive the Hacker News test?

If most of these are yes: ship it.

## Voice checklist

Before you publish any copy:

- [ ] Does the first sentence state something specific?
- [ ] Are there any weasel words ("empower," "leverage," "enable," "robust")?
- [ ] Is there any passive voice that could be made active?
- [ ] Would this sentence appear identically on a competitor's site?
- [ ] Is there any forced humor that should be removed?
- [ ] American English spelling?
- [ ] Sentence case for headings?
- [ ] En dashes ( – ) instead of em dashes (—)?
- [ ] Straight quotes instead of curly quotes?

## Common mistakes

| Mistake                         | Fix                                                      |
| ------------------------------- | -------------------------------------------------------- |
| Generic SaaS gradients          | Solid backgrounds or PostHog-certified gradients         |
| Stock illustrations             | Custom illustration                                      |
| Abstract blobs                  | Real visual elements with purpose                        |
| "Empowers teams to leverage..." | Just say what it does                                    |
| Forcing humor                   | Be clear; let wit emerge naturally                       |
| Perfect but hollow design       | Intentional and human                                    |
| Logo on a busy background       | Use a clear container                                    |
| AI hedgehog art                 | [Submit an art request](/handbook/brand/art-requests) |
| Copying competitor aesthetics   | Ask: does this feel like PostHog?                        |

## "This or that" quick guide

| Scenario                             | Use this                                          |
| ------------------------------------ | ------------------------------------------------- |
| Logo on dark background              | White/light logo variant                          |
| Logo on light background             | Standard or dark logo variant                     |
| Expressive display text with hog art | Squeak                                            |
| All other typography                 | Open Runde                                        |
| Hedgehog artwork                     | Monoline, beige body, brown spines, no highlights |
| Icon style (product UI)              | Central Icon System (outlined vectors)            |
| Icon style (casual/stickers)         | icons8 sticker pack                               |
| Shadows                              | Hard drop shadow, `#000000` black                 |
