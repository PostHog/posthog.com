## What is the job to be done?

"Help me understand how my AI features perform, what they cost, and how users interact with them."

- Track model performance: latency, cost per query, token usage, error rates
- Evaluate AI output quality and detect regressions after prompt or model changes
- Understand how users actually interact with AI-generated output
- A/B test prompts, models, and parameters against real user behavior metrics
- Monitor cost attribution by user, organization, or feature

## What PostHog products are relevant?

- **[LLM Observability](/docs/ai-engineering) (core)** — Model performance, cost tracking, latency monitoring
- **AI Evals** — Score and evaluate AI outputs, surface quality issues proactively
- **[Product Analytics](/docs/product-analytics)** — User behavior around AI features
- **[Experiments](/docs/experiments)** — A/B test prompts, models, parameters
- **[Error Tracking](/docs/error-tracking)** — Catch model failures, hallucinations, timeouts
- **[Session Replay](/docs/session-replay)** — See the user experience of AI features
- **[PostHog AI](/docs/posthog-ai/allow-access)** — Query model performance data, cost analysis
